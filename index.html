Основные узлы

Кормушка

Представляет собой деревянную конструкцию, среднего размера. Имеются отверстия с трех сторон для свободного передвижения птиц внутри кормушки.
Подача корма производится автоматически, в заданное время.

Бункер для корма

Выполнен из фанеры, представляет собой мотор-редуктор и лопасть, при вращении которой корм начинает сыпаться из бункера.

Камера

Установлена внутри кормушки. Позволила наблюдать за посетителями в режиме реального времени. Наблюдать можно с любого устройства.

Дозатор

Выполнен из фанеры, представляет собой мотор-редуктор и лопасть, при вращении которой корм начинает сыпаться из бункера.
